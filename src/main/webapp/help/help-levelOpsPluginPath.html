The work directory where the propelo plugin will store permanent and intermidiate information. The jenkins process must have write access on this directory.

NOTE:
The Propelo Plugin directory can contain environment variables. The format is ${ENV_VAR}.

Preferred location is ${JENKINS_HOME}/propelo-job-reporter-plugin

e.g. /var/jenkins_home/propelo-job-reporter-plugin or /var/lib/jenkins/propelo-job-reporter-plugin
