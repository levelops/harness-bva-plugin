Configuration for Build Jobs in JSON format. A build job can be specified in two ways: 1) Specify the job name. e.g. Job Name is "Build some code". 2) Specify the job name & job parameter. e.g. Job Name is "Build some code" and Job Param Name is "branch" and Job Param Value is "dev".

NOTE:
The job name, param name & param value all support regex.

Example: [{"jobName": "name1"}, {"jobName": "name1", "paramName": "branch", "paramValue": "dev"}]
