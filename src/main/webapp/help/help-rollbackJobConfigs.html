Configuration for Rollback Jobs in JSON format. A rollback job can be specified in two ways: 1) Specify the job name. e.g. Job Name is "Deploy some code". 2) Specify the job name & job parameter. e.g. Job Name is "Deploy some code" and Job Param Name is "rollback" and Job Param Value is "true".

NOTE:
The job name, param name & param value all support regex.

Example: [{"jobName": "deploy service 1"}, {"jobName": "deploy service 2", "paramName": "rollback", "paramValue": "true"}]
