A job can be specified in one of two ways: 1) Specify the job name. e.g. Job Name is "Deploy some code". 2) Specify the job parameter name & job parameter value. e.g. Job Name is "Deploy some code" and Job Param Name is "region" and Job Param Value is "us-west".

NOTE:
The job name, param name & param value all support regex.

By default job parameter name & parameter value are non needed.
